# Whack-A-Mole
 Whack A Mole game built in React Native
 
 
 ## Technologies used:
 - React Native
 - Node.js 
 - Heroku 
 - MongoDB atlas
 
 
 
 ## Note:
 
 Backend is containerized and deployed live via heroku container registry.
 
 You may test via a GET request to https://whack-backend.herokuapp.com/api/test
